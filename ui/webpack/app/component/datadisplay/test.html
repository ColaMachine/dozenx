2、如何在开始菜单中添加未经过安装的绿色软件

▼那应该如何设置呢？首先是取消掉原有的不知道什么玩意的模块，右键点击相应的模块，点“从开始屏幕取消固定”即可取消。

▼接着，从左侧的列表中拖到即可。里面图标的大小都是可以设置的，想多放就缩小些。

▼这个其实很简单，复杂的部分在于，你直接从桌面或者文件夹拖图标，是进不到开始菜单里的。这种情况主要影响的是绿色软件无法进入到这里。这时就需要一定的设置了，具体的方式是这样：

打开你自己电脑的开始菜单文件夹，一般的位置在    

C:ProgramDataMicrosoftWindowsStart MenuPrograms  

▼接着找到你想要放到开始菜单里的绿色软件，右键点击并创建快捷方式，接着将快捷方式拷贝到刚才的文件夹中，再打开开始菜单，就可以在左侧的列表中看到该程序的图标了，这时通过拖动就能将这个软件固定到开始菜单了 。

3、如何备份已经设置好的开始菜单

经过上面的设置，我们可以打造出最适合自己使用习惯的开始菜单，虽然是很小的改动，但用熟之后，确实可以很大程度的提升效率。但有时候我们需要重装系统，开始菜单重新再设置一次，是非常费劲的。这时，我们需要一款软件来实现，软件名为：Backup Start Menu Layout

下载地址：点这里。

▼软件打开后，就是这个界面，第一步，点击红框中的图标，创建条目

▼创建好后，可以改个自己记得住的名字，改好之后选择，点击时钟图标即可备份。

▼备份好后，无论你把开始菜单打得再乱，双击上面的条目即可恢复。

不过要注意，这里的备份与恢复，是建立在刚才提到的“开始菜单文件夹”中已经有对应快捷方式的情况，如果没有是无法恢复的。所以建议备份时一方面要将已经设置好的“开始菜单文件夹”也备份一下，并在重装系统后拷贝到原位置；另一方面，也要注意不要移动软件存放的位置，以免空有图标打不开软件 。

四、如何保证系统安全

1、卸载掉一切“XX管家”

其实这个是很多电脑小白的常见状况，就是务必先装一个XX电脑管家 。这个管家除了每天提醒你开机用了多少秒、显示个破泡泡告诉你电脑占用了多少内存、假装能帮你修复电脑问题等毫无用处的功能之外，倒是既能拖慢电脑速度又包安装各种全家桶 。

所以我个人的建议，第一步就是先把这些类似的流氓软件都卸载掉 。

2、经常更新漏洞

时常更新漏洞是系统防御最新的攻击手段和病毒最简单、最有效的方式了。

这里其实无需什么软件，点击  设置——更新和安全，保证漏洞始终更新就可以了。同时注意调整一下windows自动更新的情况，避免使用时自动更新系统。
